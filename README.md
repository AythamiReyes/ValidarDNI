# ValidarDNI
Esta clase implementa un validador de contraseñas basado en una expresión regular.
La validación comprueba que la contraseña introducida cumpla los siguientes criterios:
Contiene al menos un número
Contiene al menos una letra mayúscula
Contiene al menos una letra minúscula
Contiene al menos un carácter especial
Tiene una longitud mínima de 8 caracteres
